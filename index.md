# Unity 扫盲手册

本系列文章是我作为 Unity 新手，希望提升知识深度和广度所进行的补盲学习笔记，而不是Unity新手教程。适用于这样的读者：

- 已经熟悉 Unity 编辑器，已经有能力编辑UI、场景
- 熟悉 C#、Lua，已经能够完成模块开发

Unity 闭源以及官方文档的不充分给深入理解 Unity 带来了很大困难，一些技术内幕往往只能从熟练开发者的博客来获取，这对新手是极其不友好。本系列文章重点关注那些官方文档没有提及或者描述过于简略的技术细节，通过 AI 辅助搜索大量网络文章，从碎片化的信息中获取一些有用的知识。

也许 AI 给出的总结仍是模棱两可的，但它给出的信息中仍包含了可供自行探索的出发点，例如一些技术名词、论坛连接、Github 仓库等等。作为已经具备基础研究能力的开发者，我们可以从这里开始进行人工学习，自行理解、总结。

Unity 涉及技术众多，本人不可能全盘掌握，而是以客户端程序的视角、以普通职场员工的工作涉及范围进行搜集和归纳。希望这些文章能帮助新人建立基础的知识体系，也希望帮助本人自己更好从事未来工作。
